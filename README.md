# AV-data-android-app

## ⚠️ Git LFS Required
This repository uses Git Large File Storage (LFS) for Gradle cache files.

**To clone and use this repository:**

1. Install Git LFS: https://git-lfs.github.com/
2. Run: `git lfs install`
3. Clone normally: `git clone https://github.com/YuyueminAustin/AV-data-android-app.git`
4. Or after cloning: `git lfs pull` to download all LFS files

Without Git LFS, the large files will appear as small pointer files.
